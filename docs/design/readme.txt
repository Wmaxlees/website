some technologies
the MEAN stack: MongoDB, Express, Angular, Node:

https://nodejs.org/ - server-side javascript 
http://expressjs.com/ - popular node.js web app framework
https://www.mongodb.org/ - object-relational database platform built on node.js
https://angularjs.org/ - javascript library designed by google that works with html to build dynamic websites
https://angular.github.io/protractor/ - testing framework built on selenium and jasmine, designed by google to work with angular websites

other useful technologies:
http://getbootstrap.com/2.3.2/ - twitter bootstrap, a web development framework for easy addition of common web elements

what to install:
install node.js: https://nodejs.org/
install mongoDb: https://www.mongodb.org/
install protractor: https://angular.github.io/protractor/
clone this repository

what to do:
are you working on the front-end content?  spend 16 hours learning angular.js at your favorite learning site or try a recommendation below
are you working on the server-side content?  spend 16 hours learning mongodb and node.js
do you want to learn more?  get TDD/BDD with us.  learn protractor.

some learning sites
http://www.w3schools.com/ - reference for all things webdev
http://www.codecademy.com/ - html/css, javascript, angular.js
